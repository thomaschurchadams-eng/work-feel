# CreditUnionAI News Image Guidelines

## Purpose
Create consistent article images that make the site feel credible, modern, and executive-facing — not gimmicky AI art.

## Default Image Standard
- Format: JPG for article hero/card images
- Preferred size: 1600×900 or 1600×896
- Acceptable: 1536×1024 when article layout needs a taller crop
- Style: professional editorial illustration / realistic business-tech visual
- Avoid: distorted hands/faces, fake logos, readable fake text, brand logos we do not have rights to use, scary robot clichés

## Visual Direction
CreditUnionAI News should feel:
- trustworthy
- boardroom-ready
- financial-services credible
- clear, practical, and non-hype

Good motifs:
- credit union branch / boardroom / operations center
- abstract AI network overlays
- fraud/risk dashboards without readable sensitive data
- member service/contact center scenes
- lending/document workflows
- payments/authentication visuals
- governance/compliance review scenes

## Automated Image Workflow
1. Create or select an image that matches the article and publication style.
2. Save it in `/assets/` with a descriptive lowercase kebab-case filename.
3. Add accurate alt text and social metadata.
4. Validate that it contains no fake readable text, unauthorized logos, confidential data, distorted people, or misleading product representations.
5. Include it in the same production package as the article. Routine images do not require human approval.

## Prompt Template
Create a professional editorial image for a financial services publication about [TOPIC]. Show [SCENE]. Visual style: modern, credible, clean, executive-facing, realistic but lightly stylized. Include subtle abstract AI/data network elements. No readable text, no logos, no brand marks, no distorted faces or hands. Color palette: navy, blue, white, muted gold accents. Aspect ratio 16:9.

## File Naming
Use lowercase kebab-case when adding new files:
- `assets/ai-governance-board-briefing.jpg`
- `assets/fraud-detection-dashboard-credit-union.jpg`
- `assets/member-messaging-ai-trust.jpg`

Note: existing repo has mixed file naming; new files should be cleaner going forward.
