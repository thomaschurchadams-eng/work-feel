# Credit Union AI Media Platform

## Preview locally
1. Start a simple static server from the project root:
   ```bash
   python -m http.server 8000
   ```
2. Open the site in your browser at http://localhost:8000/ to view `index.html` with `styles.css`.

## Project files
- `index.html` – Landing page markup for the Credit Union AI Media Platform.
- `styles.css` – Styling for the landing page sections, typography, and components.
